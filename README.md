Description
===================
yaml2json 

Feature
====================
* zero config.
* zero install.
* support windows,linux,Mac os,freebsd,netbsd,openbsd,plan9 platform

Usage
====================
### shell
* find the build of you platform
* run `echo "a: 1" | yaml2json` to see result

### read from file save to file
```
cat 1.yml | yaml2json > 2.json
```
